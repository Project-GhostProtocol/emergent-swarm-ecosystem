Emergent Swarm Ecosystem

An artificial life sandbox built with p5.js.

Features:
Swarm intelligence
Predator pack hunting
Apex predator system
Infection spread
Evolutionary reproduction
Dynamic food economy
Cinematic visual effects
Emergent ecosystem behavior

Every entity follows simple rules, but together they create unpredictable large-scale behavior and survival dynamics.

Press K to wipe the prey population and trigger the apex predator event.
